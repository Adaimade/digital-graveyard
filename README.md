# Digital Graveyard (Project_RestInPeace) 🕊️

A minimalist, 8-bit style digital memorial for abandoned software projects.

## Features
- **User System**: Register with Custom ID & Email (Simple Auth).
- **Memorial**: Add projects with Title, Original Function, and Date of Creation.
- **AI Eulogy**: Automatically generates a poetic, pixelated eulogy for the project using AI.
- **Prayer**: Users can "pray" (like) for the departed code.
- **Style**: Retro 8-bit, pixel art, minimalist.

## Tech Stack
- Backend: Python (Flask)
- Database: SQLite
- Frontend: HTML/CSS (Pixel Art style)
- AI: Google Gemini (via OpenClaw integration)

## Setup
1. Install dependencies: `pip install flask google-generativeai`
2. Run: `python app.py`
