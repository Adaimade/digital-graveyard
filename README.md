# ⚰️ Digital Graveyard (Project_RestInPeace)
> *"Where dead code goes to rest."*

## 📜 About the Crypt Keeper (關於管理員)

**Made with 🖤 by ADAIMADE & OpenClaw斜霸**

這不僅僅是一個軟體專案，這是 **ADAIMADE** 與他的 AI 助手 **OpenClaw斜霸** 共同打造的一座數位紀念碑。

在這個快速迭代的時代，無數的代碼在被寫下的那一刻便註定被遺忘。有些專案胎死腹中，有些因為架構過度複雜而崩塌，有些則是在解決了特定問題後便失去了存在的意義。

**我們不忍心看著這些「心血」消失在 `rm -rf` 的指令中。**

於是，我們建立了這座 **Digital Graveyard**。在這裡，每一行逝去的代碼都能擁有一塊屬於它的像素墓碑，以及一段由 AI 吟唱的 8-bit 輓歌。

---

## 🕊️ Features (功能)
- **Pixel Memorials**: 每一座墓碑都以 8-bit 像素風格呈現，致敬那個純粹的時代。
- **AI Eulogies**: 內建 Gemini AI，自動為每個專案生成獨一無二的詩意悼詞。
- **Mourning System**: 訪客可以註冊、登入，並為逝去的專案獻上祈禱 (🙏 Pray)。
- **Cloud Eternal**: 部署於 Zeabur，確保這些記憶不會因本地硬碟損壞而消失。

## 🛠️ Tech Stack (技術棧)
- **Backend**: Python (Flask)
- **Database**: PostgreSQL (Production) / SQLite (Dev)
- **AI**: Google Gemini (1.5 Flash / Pro)
- **Frontend**: HTML5 + CSS3 (Retro Pixel Art Style)
- **Server**: Gunicorn

## 🕯️ How to Deploy (如何部署)
1. Clone this repo: `git clone https://github.com/Adaimade/digital-graveyard.git`
2. Set env vars: `GEMINI_API_KEY`, `SECRET_KEY`, `DATABASE_URL`
3. Push to **Zeabur** and let the spirits rise.
